FTB-py
======

Stuff for the FTB Wiki

navitems.py
-----------
This will turn a list of names in work/input.txt to NI templates. It will ask you for the mod abbreviation for use with the tilesheet extension. For example, 'Item' with abbreviation 'EX' will become '   -->{{NI|Item|mod=EX}}{{,}}<!--'. input.txt will not be replaced, instead output.txt will be created.
